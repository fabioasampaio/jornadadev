Str() → Função, pois recebe um número e retorna uma string correspondente ao seu valor.
QOut() → Procedimento, pois apenas exibe um texto na tela e não retorna um valor.
Date() → Função, pois retorna a data atual do sistema.
Len() → Função, pois retorna a quantidade de caracteres de uma string ou de elementos de um array.
Upper() → Função, pois retorna uma nova string com todos os caracteres em maiúsculas.
ClearScreen() → Procedimento, pois apenas limpa a tela, executando uma ação sem retornar um valor.
